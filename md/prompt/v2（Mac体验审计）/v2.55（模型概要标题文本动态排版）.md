# v2.55 模型概要标题文本动态排版

## 版本号与分配依据

- 版本号：v2.55。
- 分配依据：`update_log.md` 最新正式轮次为 v2.54，Agent C 已验收 `1f9986a` / run `29165271962`。

## 背景

模型页 `ModelSummaryPanel` 已有整体辅助语义，但模型名称仍使用固定 19pt 并依赖 `minimumScaleFactor` 单行压缩。iPad split view、Mac Catalyst 窄窗口和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `ModelSummaryTextLayoutPolicy`。
- 模型名称与简介改用 Dynamic Type 语义字体；名称允许两行，简介允许多行，移除名称 `minimumScaleFactor`。
- 新增 `testModelSummaryTextLayoutPolicySupportsDynamicTypeRows`（97 -> 98）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变 `ModelSummaryAccessibilityMetadata`、模型选择/部署、文件、runtime 或 verified 门禁。

## 实现要点

1. 策略：`titleSummarySpacing`、`nameLineLimit=2`、`summaryLineLimit=4`、`summaryLineSpacing`。
2. 名称 `.title3.weight(.heavy)`；简介 `.footnote.weight(.medium)`。
3. fixedSize vertical；不使用 minimumScaleFactor。

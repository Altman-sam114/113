# v2.43 / 运行策略开关行触控目标

## 版本号和分配依据

- 版本号：v2.43。
- 分配依据：`update_log.md` 最新正式版本为 v2.42，本轮继续 Agent X 总目标“优化 UI、Mac、iPad 版”的下一项独立可验证小目标。

## 背景

当前设置页和优化 dashboard 已通过 `OptimizationToggleAccessibilityMetadata` 锁住运行策略开关的辅助语义，并通过 `OptimizationToggleGridLayoutPolicy` 锁住窄屏单列、iPad/Mac 宽区域双列的网格策略。但单个 `OptimizationToggleRow` 只依赖内部 padding 形成命中区域，没有独立 policy/test 锁住 44pt 触控目标；后续 UI 调整可能意外压缩开关行高度，影响 Mac Catalyst、iPad 触控和辅助技术用户的可用性。

## 目标

- 新增运行策略开关行 44pt 触控目标策略。
- 设置页和优化 dashboard 的单个运行策略开关行保持至少 44pt 命中高度。
- 新增测试锁住最小触控目标、行最小高度映射和策略达标判断。
- 同步 README、测试规范、核心流程、流程图、入口规则、更新日志和本提示词归档。

## 非目标

- 不修改 `DeviceOptimizer` 默认开关状态或切换逻辑。
- 不修改运行策略顺序、标题、说明文案或指标数据。
- 不修改 `OptimizationToggleAccessibilityMetadata` 的 label/value/hint/input labels/identifier。
- 不修改 `OptimizationToggleGridLayoutPolicy` 的列数、间距或断点。
- 不修改芯片准备度动态摘要。
- 不修改模型文件、artifact 校验、runtime plan 或 verified 门禁。
- 不接入真实 runtime，不下载模型权重，不发送到云端服务。

## 当前架构依据

- `md/flow/flow.md` 规定运行策略开关只切换本地运行策略，不下载模型权重、不启动真实 runtime、不发送云端服务。
- `LocalGemma/ContentView.swift` 中 `OptimizationToggleRow` 同时被设置页和优化 dashboard 的 `OptimizationToggleGrid` 复用。
- `LocalGemmaTests/LocalGemmaTests.swift` 已有 `testOptimizationToggleAccessibilityMetadataDescribesLocalOnlyPolicy` 和 `testOptimizationToggleGridLayoutPolicyUsesTwoColumnsOnWideSettingsWidth`。
- 项目已有 `HeaderActionLayoutPolicy`、`SettingsIconActionLayoutPolicy`、`SessionBarActionLayoutPolicy` 等 44pt 触控目标策略可作为实现模式。

## 实现步骤

1. 在 `LocalGemma/ContentView.swift` 新增 `OptimizationToggleRowLayoutPolicy`：
   - `minimumTouchTarget = 44`
   - `rowMinHeight = minimumTouchTarget`
   - `usesMinimumTouchTarget()` 返回 `rowMinHeight >= minimumTouchTarget`
2. 调整 `OptimizationToggleRow`：
   - 在按钮内容上使用 `.frame(minHeight: OptimizationToggleRowLayoutPolicy.rowMinHeight)`。
   - 保留原有 `.padding(12)`、背景、buttonStyle、accessibility metadata、toggle state flow 和 grid layout。
3. 在 `LocalGemmaTests/LocalGemmaTests.swift` 新增 `testOptimizationToggleRowLayoutPolicyMaintainsTouchTarget`：
   - 断言 `minimumTouchTarget == 44`。
   - 断言 `rowMinHeight == minimumTouchTarget`。
   - 断言 `usesMinimumTouchTarget()` 为 true。
   - 断言 `rowMinHeight >= minimumTouchTarget`。
4. 同步文档：
   - `AGENTS.md`
   - `README.md`
   - `md/test/test.md`
   - `md/flow/flow.md`
   - `md/flow/flowchart.md`
   - `update_log.md`

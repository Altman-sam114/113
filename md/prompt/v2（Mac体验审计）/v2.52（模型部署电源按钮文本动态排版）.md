# v2.52 模型部署电源按钮文本动态排版

## 版本号与分配依据

- 版本号：v2.52。
- 分配依据：`update_log.md` 最新正式轮次为 v2.51，Agent C 已验收 `9ee9099` / run `29164563415`。

## 背景

模型页 `DeploymentPowerButton` 已有 44pt 触控目标和完整辅助语义，但主标题仍使用固定 21pt 字号并依赖 `minimumScaleFactor` 单行压缩，副标题也是固定 12pt。iPad split view、Mac Catalyst 窄窗口和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `ModelDeploymentPowerTextLayoutPolicy`。
- 主标题与副标题改用 Dynamic Type 语义字体。
- 主标题允许两行，副标题允许两行；移除主标题 `minimumScaleFactor`。
- 新增 `testModelDeploymentPowerTextLayoutPolicySupportsDynamicTypeRows`（94 -> 95）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变 `ModelDeploymentControlLayoutPolicy` 触控目标、部署启停状态流、辅助语义、verified 门禁。
- 不下载模型权重，不接真实 runtime，不创建原生 macOS target。

## 实现要点

1. 策略：`verticalSpacing`、`titleLineLimit=2`、`subtitleLineLimit=2`。
2. 主标题 `.title3.weight(.heavy)`；副标题 `.footnote.weight(.bold)`。
3. fixedSize vertical；不使用 minimumScaleFactor。
4. 测试锁住策略与多行能力，并确认电源按钮最小高度策略仍 >= 44。

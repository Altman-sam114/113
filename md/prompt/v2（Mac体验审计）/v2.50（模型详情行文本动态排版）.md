# v2.50 模型详情行文本动态排版

## 版本号与分配依据

- 版本号：v2.50。
- 分配依据：`update_log.md` 最新正式轮次为 v2.49，Agent C 已验收 `65cbb59` / run `29164006779`。

## 背景

模型页 `DetailRow` / `AdviceRow` 已有行级辅助语义，但标题/数值/建议文本仍使用固定小字号，`DetailRow` 还依赖 `minimumScaleFactor` 压缩。iPad split view、Mac Catalyst 窄窗口和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `ModelDetailRowTextLayoutPolicy`。
- `DetailRow` 标题/数值、`AdviceRow` 建议文本改用 Dynamic Type 语义字体。
- 移除 `DetailRow` 的 `minimumScaleFactor` 压缩；标题允许两行，数值允许两行，建议允许多行。
- 新增 `testModelDetailRowTextLayoutPolicySupportsDynamicTypeRows`（92 -> 93）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变 `ModelDetailRowAccessibilityMetadata`、模型详情列宽、选择/部署/文件/runtime/verified 门禁。
- 不下载模型权重，不接真实 runtime，不创建原生 macOS target。

## 实现要点

1. 策略常量：`horizontalSpacing`、`titleLineLimit=2`、`valueLineLimit=2`、`adviceLineLimit=4`、`adviceLineSpacing`、`minimumRowHeight`。
2. 标题 `.caption.weight(.bold)`，数值 `.subheadline.weight(.black)`，建议 `.subheadline.weight(.semibold)`。
3. vertical fixedSize，不使用 minimumScaleFactor。
4. 测试锁住策略与多行能力。

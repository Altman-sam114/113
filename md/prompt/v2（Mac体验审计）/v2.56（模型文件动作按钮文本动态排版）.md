# v2.56 模型文件动作按钮文本动态排版

## 版本号与分配依据

- 版本号：v2.56。
- 分配依据：`update_log.md` 最新正式轮次为 v2.55，Agent C 已验收 `4013446` / run `29165656595`。

## 背景

模型页 `ArtifactActionButton` 已有辅助语义与文件工作流入口，但标题/副标题仍使用固定小字号并依赖 `minimumScaleFactor` 单行压缩。iPad split view、Mac Catalyst 窄窗口和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `ModelArtifactActionTextLayoutPolicy`。
- 标题与副标题改用 Dynamic Type 语义字体，允许两行，移除 `minimumScaleFactor`。
- 新增 `testModelArtifactActionTextLayoutPolicySupportsDynamicTypeRows`（98 -> 99）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变模拟暂存/卸载确认/扫描/导入状态流、artifact 校验、辅助语义、verified 门禁。

## 实现要点

1. 策略：`verticalSpacing`、`titleLineLimit=2`、`subtitleLineLimit=2`、`minimumHeight`。
2. 标题 `.subheadline.weight(.black)`；副标题 `.caption2.weight(.bold)`。
3. fixedSize vertical；不使用 minimumScaleFactor。

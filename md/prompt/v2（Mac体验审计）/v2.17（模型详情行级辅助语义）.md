# v2.17 / 模型详情行级辅助语义

## 版本号和分配依据

- 版本号：`v2.17`
- 分配依据：`update_log.md` 最新正式版本为 `v2.16 / 模型状态徽章辅助语义`，本轮继续递增到 `v2.17`。
- 所属总目标：Agent X 持续优化 UI、Mac 和 iPad 体验。

## 背景

模型页已经具备模型选择器、模型状态徽章、部署控件和模型详情右栏整体辅助语义。Mac/iPad 宽屏模型页中，`ModelDetailColumn` 内的 `DetailRow` 和 `AdviceRow` 是用户逐行扫描参数、性能和运行建议的高频信息单元，但目前它们只有视觉文本和图标，没有行级 VoiceOver / Voice Control 入口。整体详情摘要适合快速概览，行级语义适合逐项确认。

## 目标

1. 为模型详情参数行、性能行和建议行建立统一的行级辅助语义。
2. 让 `DetailRow` 暴露 label、value、hint、Voice Control 输入标签和稳定 identifier。
3. 让 `AdviceRow` 区分运行阻塞项、下一步建议和芯片策略建议，并暴露 label、value、hint、Voice Control 输入标签和稳定 identifier。
4. 文案必须明确行级内容只展示本地模型详情，不下载模型权重、不启动真实 runtime、不发送云端服务、不绕过 artifact verified 门禁。
5. 新增 XCTest 覆盖参数行、性能行、建议行分类、identifier、Voice Control input labels 和安全边界。
6. 同步 `AGENTS.md`、`README.md`、`md/flow/flow.md`、`md/flow/flowchart.md`、`md/test/test.md` 和 `update_log.md`。

## 非目标

- 不修改 `ModelCatalog`、`LocalArtifactValidator`、`LocalRuntimePlanner` 或部署状态流。
- 不改变模型页布局、视觉样式、颜色或断点。
- 不修改 `ModelDetailAccessibilityMetadata` 的整体摘要语义，除非为避免重复播报需要做最小兼容调整。
- 不新增 UI Test target。
- 不下载模型权重，不接入云端推理，不接入真实 runtime，不创建原生 macOS target。
- 不绕过 artifact `verified` 门禁。

## 当前架构依据

- `LocalGemma/ContentView.swift`
  - `ModelDetailColumn` 汇总 `ModelSummaryPanel`、`ModelParametersPanel`、`ModelPerformancePanel` 和 `ModelAdvicePanel`。
  - `DetailRow` 展示模型参数和性能键值行。
  - `AdviceRow` 展示运行阻塞项、下一步建议和芯片策略建议。
  - 已有同类模式：`ModelDetailAccessibilityMetadata`、`ModelStatusBadgeAccessibilityMetadata`、`OptimizerMetricAccessibilityMetadata`。
- `LocalGemmaTests/LocalGemmaTests.swift`
  - 当前测试基线为 60 个 XCTest 方法。

## 实现步骤

1. 在 `ContentView.swift` 中新增 `ModelDetailRowAccessibilityMetadata`。
2. 元数据建议包含：
   - `static let hint`
   - `static func label(title:)`
   - `static func value(title:value:)`
   - `static func inputLabels(title:)`
   - `static func identifier(title:)`
   - `enum AdviceKind`
   - `static func adviceLabel(kind:)`
   - `static func adviceValue(text:)`
   - `static func adviceInputLabels(kind:)`
   - `static func adviceIdentifier(kind:sequence:)`
3. 将 `DetailRow` 设置为独立只读辅助元素。
4. 为 `AdviceRow` 增加明确的 `kind` 参数，并在 `ModelAdvicePanel` 的 blocker、next step 和芯片策略建议处传入对应 kind。
5. 不改变 row 的视觉文本、图标、颜色或布局。
6. 在 `LocalGemmaTests.swift` 新增 `testModelDetailRowsExposeAccessibilityMetadata`，测试函数数从 60 增加到 61。

## 风险和禁止项

- 风险：模型详情区已有整体摘要，行级语义不能导致明显重复播报；如需调整，应保证整体摘要仍可测试。
- 风险：identifier 必须稳定，不能依赖动态模型名、prompt 或完整建议文本。
- 禁止下载模型权重、云端推理、绕过 verified 门禁。
- 禁止把未跑测试写成已通过。
- 禁止验收旧 run 或旧 artifact。

# v2.58 工作区侧栏文本动态排版

## 版本号与分配依据

- 版本号：v2.58。
- 分配依据：`update_log.md` 最新正式轮次为 v2.57，Agent C 已验收 `ca42cbe` / run `29166318654`。

## 背景

大屏 `sidebarTabPicker` 已有 44pt 触控目标与辅助语义，但标题/副标题仍使用固定字号，副标题还依赖 `minimumScaleFactor` 单行压缩。iPad/Mac regular 侧栏和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `WorkspaceSidebarTextLayoutPolicy`。
- 工作区标题与 detailed 副标题改用 Dynamic Type 语义字体；标题允许两行，副标题允许两行，移除副标题 `minimumScaleFactor`。
- 新增 `testWorkspaceSidebarTextLayoutPolicySupportsDynamicTypeRows`（100 -> 101）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变 `WorkspaceTab.shortcutKey`、command menu、`selectedTab`、composer focus、导航 44pt 触控目标或辅助语义。

## 实现要点

1. 策略：`titleSubtitleSpacing`、`titleLineLimit=2`、`subtitleLineLimit=2`。
2. 标题 `.subheadline.weight(.black)`；副标题 `.caption.weight(.semibold)`。
3. fixedSize vertical；不使用 minimumScaleFactor。

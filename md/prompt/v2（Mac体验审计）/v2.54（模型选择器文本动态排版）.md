# v2.54 模型选择器文本动态排版

## 版本号与分配依据

- 版本号：v2.54。
- 分配依据：`update_log.md` 最新正式轮次为 v2.53，Agent C 已验收 `8551310` / run `29165018813`。

## 背景

模型页 `ModelSelectorPanel` 已有 44pt 触控目标和辅助语义，但选中模型名与规格摘要仍使用固定字号，并依赖 `minimumScaleFactor` 单行压缩。iPad split view、Mac Catalyst 窄窗口和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `ModelSelectorTextLayoutPolicy`。
- 选中模型名与规格摘要改用 Dynamic Type 语义字体，允许两行，移除 `minimumScaleFactor`。
- 新增 `testModelSelectorTextLayoutPolicySupportsDynamicTypeRows`（96 -> 97）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变模型选择状态流、`ModelDeploymentControlLayoutPolicy` 触控目标、部署/文件/runtime/verified 门禁或辅助语义。

## 实现要点

1. 策略：`verticalSpacing`、`nameLineLimit=2`、`specLineLimit=2`。
2. 名称 `.headline.weight(.black)`；规格 `.caption.weight(.semibold)`。
3. fixedSize vertical；不使用 minimumScaleFactor。
4. 测试锁住策略、多行能力，并确认选择器最小高度策略仍 >= 44。

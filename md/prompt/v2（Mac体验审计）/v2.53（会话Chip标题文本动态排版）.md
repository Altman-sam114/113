# v2.53 会话 Chip 标题文本动态排版

## 版本号与分配依据

- 版本号：v2.53。
- 分配依据：`update_log.md` 最新正式轮次为 v2.52，Agent C 已验收 `8489771` / run `29164782546`。

## 背景

推理页 `SessionChip` 已有选择/删除 44pt 触控目标和辅助语义，但会话标题仍使用固定 12pt 并依赖 `minimumScaleFactor` 单行压缩。iPad split view、Mac Catalyst 窄窗口和较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `SessionChipTextLayoutPolicy`。
- 会话标题改用 Dynamic Type 语义字体，允许两行，移除 `minimumScaleFactor`。
- 新增 `testSessionChipTextLayoutPolicySupportsDynamicTypeTitles`（95 -> 96）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变会话选择/删除状态流、`SessionChipActionLayoutPolicy` 触控目标、composer 聚焦、辅助语义、模型文件或 verified 门禁。

## 实现要点

1. 策略：`titleLineLimit=2`。
2. 标题 `.footnote.weight(.black)` + fixedSize vertical。
3. 测试锁住策略、多行能力和与 44pt 选择触控目标共存。

# v2.57 导出弹层标题文本动态排版

## 版本号与分配依据

- 版本号：v2.57。
- 分配依据：`update_log.md` 最新正式轮次为 v2.56，Agent C 已验收 `6c6b1c0` / run `29165957345`。

## 背景

导出弹层 `exportHeader` 已有宽屏内容宽度与分享/复制 44pt 策略，但会话标题仍使用固定 17pt 并依赖 `minimumScaleFactor` 单行压缩。iPad/Mac 宽 sheet 与较大 Dynamic Type 下可读性不足。

## 目标

- 新增 `ExportSessionTitleTextLayoutPolicy`。
- 导出会话标题与消息摘要改用 Dynamic Type 语义字体；标题允许两行，摘要允许两行，移除标题 `minimumScaleFactor`。
- 新增 `testExportSessionTitleTextLayoutPolicySupportsDynamicTypeRows`（99 -> 100）。
- 同步文档；完整验证走 GitHub Actions，禁止本机 Xcode/Simulator。

## 非目标

- 不改变 ExportPayload、ShareLink 文件优先/文本兜底、剪贴板、分享/复制触控目标、导出弹层辅助语义或会话状态流。

## 实现要点

1. 策略：`verticalSpacing`、`titleLineLimit=2`、`metaLineLimit=2`。
2. 标题 `.headline.weight(.black)`；摘要 `.footnote.weight(.bold)`。
3. fixedSize vertical；不使用 minimumScaleFactor。
